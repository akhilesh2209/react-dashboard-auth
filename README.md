# React Dashboard Application

A responsive dashboard application built using React.js with authentication, protected routes, and analytics views. This project demonstrates real-world frontend development practices.

## Live Demo

https://react-dashboard-auth.vercel.app/

## Repository

https://github.com/akhilesh2209/react-dashboard-auth

## Features

- User authentication (Login & Register)
- Protected routes using React Router
- Dashboard with KPI statistics
- Analytics page with chart visualization
- Settings page displaying user information
- Skeleton loaders for improved UX
- Persistent login using LocalStorage
- Responsive layout

## Tech Stack

- React.js
- JavaScript (ES6+)
- React Router DOM
- Context API
- CSS
- Vite
- Vercel
